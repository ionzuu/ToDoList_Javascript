# 📝 To-Do List App
A simple web-based to-do list built with vanilla JavaScript. You can add, view, and delete tasks in real-time.

# 🚀 Features
✅ Add tasks
🗑️ Remove tasks individually
🔁 Reset the entire list

📦 No frameworks or libraries used
